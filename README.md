# Anoop Kumar - Developer Portfolio

A modern, responsive developer portfolio built with React and Vite.

## Features

- Dark theme design
- Responsive layout for all devices
- Smooth scroll navigation
- Clean and modern UI

## Sections

- **Home**: Hero section with name, role, and tagline
- **About**: Brief introduction and experience summary
- **Skills**: Technical skills categorized by languages, automation tools, frameworks, and tools
- **Projects**: Showcase of key projects
- **Experience**: Professional experience details
- **Contact**: Contact information and links

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open [http://localhost:5173](http://localhost:5173) in your browser

## Build for Production

Run `npm run build` to build the project for production.
